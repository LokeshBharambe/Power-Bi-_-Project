# Customer Insight Dashboard using Power BI

## Project Overview
This project is an interactive **Power BI Customer Insight Dashboard** developed to analyze customer behavior, sales performance, and business trends using data visualization.

The dashboard converts raw customer transaction data into meaningful insights that help businesses make data-driven decisions.

---

## Objective
The main objective of this project is to:

- Analyze customer purchasing behavior
- Track sales performance
- Monitor key business KPIs
- Identify customer trends and patterns
- Generate actionable business insights

---

## Built With

- **Microsoft Power BI Desktop**
- **CSV Dataset**
- **Power Query**
- **DAX (Data Analysis Expressions)**
- **Data Modeling**

---

## Dashboard Features

### KPI Analysis
- Total Revenue
- Total Orders
- Average Sales
- Customer Count

### Customer Insights
- Customer Segmentation
- Purchase Trends
- Buying Behavior Analysis

### Sales Performance Analysis
- Monthly Sales Trends
- Product Category Performance
- Revenue Distribution

### Interactive Visualizations
- Slicers and Filters
- Dynamic Charts
- Drill-down Analysis
- Interactive Reports

---

## Power BI Concepts Used

- Data Cleaning using Power Query
- Data Transformation
- Data Modeling
- DAX Measures
- KPI Cards
- Interactive Dashboard Design
- Report Visualization
