## Dashboard Screenshots

### 1. Sales Overview Dashboard
Provides a high-level summary of business performance with key KPIs including:

- Total Revenue
- Total Units Sold
- Average Revenue per Sale
- Monthly Sales Trend Analysis
- Sales by Category
- Sales by Region

**Purpose:**  
Helps monitor overall business performance and sales distribution.

---

### 2. Product Details Dashboard
Focused on product-level performance analysis.

Includes:

- Product-wise Sales Comparison
- Product Revenue Analysis
- Units Sold by Product
- Detailed Transaction Table
- Product Category Performance

**Purpose:**  
Identifies top-performing products and product sales trends.

---

### 3. Customer Insights Dashboard
Analyzes customer behavior and purchasing patterns.

Includes:

- Total Customers
- New Customer Percentage
- Average Purchase Value
- Customer Type Distribution
- Payment Method Analysis
- Geographic Customer Distribution Map

**Purpose:**  
Provides customer segmentation and behavior insights.

---

### 4. Return Analysis Dashboard
Tracks product return performance.

Includes:

- Percentage of Orders Returned
- Total Returns
- Monthly Return Trends
- Return Distribution by Region
- Return Percentage by City

**Purpose:**  
Helps identify return patterns and operational issues.

---
